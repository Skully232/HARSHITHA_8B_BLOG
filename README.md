### 📝 Project Overview
This repository contains a web blog entry for **"THE DIGITAL VILLAGE: WEB DESIGN COMPETITION"** held at **Bapu Primary-High School**. 

The competition challenged students from Class 7 and 8 to apply their knowledge of **HTML and CSS** to create a digital narrative about rural roots and heritage. Harshitha's project, "Discover Ramanagara," highlights the district famously known as the "Silk City." The blog expertly balances the area's industrial importance in silk production with its world-renowned geological features and ancient history.

### 💻 Technical Skills Demonstrated
* **HTML5 Semantic Structure:** Clear use of hierarchical headers and organized paragraph blocks to separate the industrial, historical, and natural aspects of the district.
* **CSS3 Styling:** * **Layout Architecture:** Implementation of a 60% width centered container with `margin: auto`, ensuring the blog is optimized for readability.
    * **Visual UI Design:** Use of `border-radius` (10px) on the main blog-box and (8px) on images to create a cohesive, modern look.
    * **Color Theory:** Utilization of a Slate Gray background (`#55767C`) and a Peach-puff content box (`#FFDAB9`), providing a warm and professional contrast.
    * **Typography Styling:** Effective use of `<b>` (bold) and `<i>` (italics) to highlight key terms like *"Reshme Nagara"* and "monolithic granite hills."
* **Content Curation:** Comprehensive coverage of Ramanagara’s geography (Arkavati and Shimsha rivers), its archaeological importance (UNESCO standards), and its identity as a global hub for the silk trade.

3.  Open the file in any modern web browser.
